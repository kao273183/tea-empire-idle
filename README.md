# 🧋 手搖飲帝國（Tea Empire Idle）

從台灣一間小攤，蓋自動化工廠、展店、打市場戰，一路征服世界的**手搖飲放置／經營遊戲**。純前端、無需安裝，打開瀏覽器就能玩。

## ▶️ 立即遊玩

👉 **https://你的帳號.github.io/tea-empire-idle/**

> 部署後把上面網址換成你的 GitHub Pages 連結。

## 🎮 玩法概覽

- **🏭 工廠**：在大地圖上鋪輸送帶、蓋採集／調製機台，自動生產飲料。產出靠分流器分送到三個去處：
  - 🧋 **出貨口** → 中央倉 → 零售賣錢（B2C）
  - 📦 **批發站** → 便利商店／企業／外送平台大量批發單（B2B）
  - 🔬 **科技廠** → 研究點，解鎖更強科技
- **🏙️ 經營**：世界地圖展店、各區（台／日／韓）分別定價、雇店員、品質投資。
- **📊 市場戰**：各區有 AI 連鎖店跟你搶客，靠降價／拼品質／砸廣告／併購對手搶市佔；對手會擴張坐大或倒閉退場。
- **🌟 天賦**：暗黑風分支技能樹，四系各有「二擇一」流派 keystone，打造你的 build。
- **🏆 成就**：50 個里程碑，達成換永久全域加成。
- **✨ 轉型上市**：累積身價後 IPO 重開，換取永久乘數與天賦點。

## 💾 存檔

存在你瀏覽器的 localStorage，**各玩家獨立、不會互通**。換裝置／清快取會清掉，設定裡可手動匯出備份。

## 🛠️ 技術

- 純 **HTML + CSS + Vanilla JavaScript**，無框架、無打包、無後端。
- PWA：可「加到主畫面」、支援離線遊玩（需以 http/https 開啟）。

## 🚀 自己跑 / 部署

本機預覽（PWA 與存檔需用伺服器，不能直接開 `file://`）：

```bash
# 任選一種，在專案資料夾啟動本機伺服器
python3 -m http.server 8000
# 然後瀏覽器開 http://localhost:8000
```

部署到 GitHub Pages：push 到 GitHub → repo 設定 → **Pages** → Source 選 `main` / `root` → 等一兩分鐘即可。
